# Linux Home lab(Beginner Project)

## Overview 
This project was made to document my linux virtual machine setup and what I learned during the proccess. The goal is to build hands-on exprience with Linux commands, system navigation, and basic system administration.

## Environment 

- Host OS: Windows 11

- Virtualization: VirtualBox

- Guest OS: Linux (Ubuntu)

## What T Learned 

- Navigating the Linux file system 
- Using basic terminal commands 
- Installing packages
- Using Git and GitHub from Linux 
- Basic system monitoring (free, df, top)
